## Overview

**Green Earth** is an interactive **tree plantation campaign website** built using **HTML, Tailwind CSS, DaisyUI, and JavaScript**.  
It allows users to explore different **tree categories, view tree details, and add trees to a cart** to simulate supporting environmental initiatives.

## 🛠 Technologies Used

- **HTML5** – Structure of the application
- **Tailwind CSS** – Utility-first styling
- **DaisyUI** – UI components
- **JavaScript (Vanilla JS)** – Application logic and DOM manipulation
- **Font Awesome** – Icons
- **Programming Hero Open API** – Tree and category data

## 🚀 Features

- Responsive landing page
- Dynamic tree categories loaded from API
- Display trees by category
- Tree card UI with image, description, category, and price
- Add trees to cart
- Remove trees from cart
- Cart sidebar with total price calculation
- Loading spinner while fetching data
- Clean and modern UI design
